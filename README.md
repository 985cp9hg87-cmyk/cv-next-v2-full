# CV Next (v2) — layout completo + métricas

## Ejecutar
1. `npm install`
2. `npm run dev`  → http://localhost:3000
   - Dashboard KPIs: http://localhost:3000/dashboard
3. En Supabase → SQL Editor → pegar `sql/metrics.sql` (una sola vez).

## Notas
- Reemplaza `public/assets/foto.jpg` por tu foto real manteniendo el mismo nombre.
- Los CTAs tienen `data-cta="email|wsp|agenda"` para que el tracker registre los clics.
- No subas `.env.local` a GitHub.
